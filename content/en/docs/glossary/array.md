Examples:

- ```python
  days = ["Monday", "Tuesday", "Wednesday"]
  ```

- ```go
  days = []string{"Monday", "Tuesday", "Wednesday"}
  ````

Rational:

- Preferred over `list` because it is also a verb.
- Preferred over `slice` because in many languages it means "cutting an array."
